About

SD-Security is a free open-source security-orientated program for windows, it is also scripted entirely in the MS-DOS batch file language.

SDS is rich is features, 90% of them are only featured in the SDS source code. SDS is updated weekly with new features, plus new apps & plugins weekly too!
